PROJECTS

Shift Scheduler

Languages used:

Android, Java, C#

IDE:

Android Studio, Microsoft Visual Studio 2017

Description:

An android application used by employees of a company to view and set their schedule. The employer has their own software made in C# to set and upload shifts to the android application. All the data about shift and employee information is stored in Amazon Web Service cloud database. Specific algorithm was made to select the most eligible employee for a specific shift.

Online Mobile Shopping

Languages used:

Android, Java

IDE:

Android Studio

Description:

An android application for a mobile company which sells its products online. The user can see details about the mobile phone they want to purchase. User has option to add multiple items in cart and can check out later by using their debit or credit card. User was also sent SMS message about the confirmation.

Active Sync

Languages used:

Android, Java

IDE:

Eclipse

Description:

This android application gave the user option to directly connect to their Laptops or Personal Computers to transfer data between the phone and computer. It also gave user option to stream any music or video files directly on their phone without downloading it first.

Database Migration Tool

Languages used:

Core Java, MySQL, Microsoft Access, Oracle

IDE:

Netbeans, Microsoft Access, MySQL Workbench, Oracle.

Description:

Worked 2 months on the projects in which three databases were used and data between these databases were migrated from one database to another using basic core java principles. The main idea of the project was to have backup of the data in case of any system failure.
